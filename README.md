## Installation and Setup Instructions

#### Steps to follow:  

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`  or  `npm ci` - depend what you prefer

To Run Test Suite:  

`npm test`  

To Start Server:

`npm start`  

To Visit App:

`https://localhost:3000/`
